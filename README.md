## ✨ Overview

A **full-stack AI-powered expense tracking web app** that helps users monitor spending and receive **smart financial insights**.

Built with a modern stack and deployed live, this project demonstrates **real-world full-stack engineering + AI integration**.

---

## 🧠 Features

- ✅ Add & track daily expenses  
- 📊 Category-based analytics (charts)  
- 🤖 AI-generated spending insights  
- 💎 Premium animated UI (Framer Motion)  
- 🌌 Particle background effects  
- ⚡ Skeleton loaders & smooth UX  
- 🌍 Fully deployed (Vercel + Render)
